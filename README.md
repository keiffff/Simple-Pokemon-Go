## About
Randomly Appear Pokemon when you press the button

- pokemon data fetched from [PokeAPI](https://pokeapi.co/)
- using React & TypeScript
- hosted by S3 & CloudFront
- DNS records managed by Route53
- SSL sertificated by ACM
